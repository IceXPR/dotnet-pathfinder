# DOTNET-PATHFINDER

An C# sample function to find the route given an array providing the passage where 0 is blocked and 1 is drivible.  The route starts at 0,0 and ends on 2,2.

## Examples
```
map0 = [
  [1,1,1]
  [1,0,1]
  [1,1,1]
  ] = 2 paths

map1 = [
  [1,1,1]
  [0,1,1]
  [1,1,1]
] = 3 paths
```

## Execution
```bash
dotnet run
```
